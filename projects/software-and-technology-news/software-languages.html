Yazılım Dünyasında Alanlara Göre Kullanılan Diller

Bu Başlıklar:

  • Front-End (Ön yüz)

    • Bu alanda yer alan yapılar ve diller

    • HTML

      Web sitesi yapımında kullanılan iskelet yapısıdır. Yazılım dili değildir.

    • CSS

      Web sitesi yapımında yazılan düz yazılara renk ve biçim verilmesini bu sayede güzel görünmesini sağlar.

    • Javascript

      Web sitesinde uygulanacak herhangi bir işlemin yapılması için butona tıklanmasını yada
      birden fazla yerde yapılan aynı işlerin her yerde tekrar tekrar yazılmasını önlemek için
      fonksiyon yazılarak fonksiyonun kullanılması gerektiği yerde rahatça yapılacak işlerin kolayla yaptırılmasını sağlar.

    • React

      React, kullanıcıya güzel görünen bir arayüz (UI) oluşturmak için kullanılan en popüler JavaScript kütüphanesidir.
      Bu kütüphane Facebook tarafından 2013’de açık kaynaklı olarak geliştirilmiştir.
      Web sitesi yapımında arayüzde yer alan herşeyi bileşenlere ayırarak sitenin daha kolay yönetilebilir olmasını sağlar.
