Front-End Alanında Kullanılan Diller

HTML

Web sitesi yapımında kullanılan iskelet yapısıdır. Yazılım dili değildir.

CSS

Web sitesi yapımında yazılan düz yazılara renk ve biçim verilmesini bu sayede güzel görünmesini sağlar.

Javascript

Web sitesinde uygulanacak herhangi bir işlemin yapılması için butona tıklanmasını yada birden fazla yerde yapılan aynı işlerin her yerde tekrar tekrar yazılmasını önlemek için fonksiyon yazılarak fonksiyonun kullanılması gerektiği yerde rahatça yapılacak işlerin kolayla yaptırılmasını sağlar.

React

React, kullanıcıya güzel görünen bir arayüz (UI) oluşturmak için kullanılan en popüler JavaScript kütüphanesidir. Bu kütüphane Facebook tarafından 2013’de açık kaynaklı olarak geliştirilmiştir. Web sitesi yapımında arayüzde yer alan herşeyi bileşenlere ayırarak sitenin daha kolay yönetilebilir olmasını sağlar.

Vue

Ön yüz geliştirme (frontend development) için kullanılan modern ve açık kaynaklı bir JavaScript framework'üdür. İlk olarak 2014 yılında 'Evan You' tarafından geliştirilen Vue.js, özellikle kullanıcı arayüzleri ve tek sayfa uygulamaları (Single Page Application) oluşturmak için tasarlanmıştır. Hem basit hem de karmaşık projeler için uygun olan Vue.js, bileşen tabanlı yapısı ve reaktif veri bağlama özellikleriyle dikkat çeker.

Back-End Alanında Kullanılan Diller

C

C yazılım dili; C++, Java ve C# gibi diğer programlama dillerinin temelini oluşturduğu için son derece kullanışlıdır ve kolay öğrenilir. C yazılım dilinin yapısal bir programlama dili olması, karmaşık kodların bloklara bölünerek kolay bir şekilde yazılabilmesi anlamına gelmektedir.

C#

Bankacılık, finans ve analist sektöründe tercih edilen otomasyon uygulamasının yazıldığı dildir.

C++

Nesneye yönelik programlama yöntemi kullanılarak kullanıcının bilgisayarda kullanabileceği uygulamalar yapmayı sağlar.

Python

C++'a göre uygualama yapmak daha kolaydır. İstenildiğinde analiz işlemlerinde kullanılır.

Java

Bankacılık ve finans sektöründe tercih edilen otomasyon uygulamasının yazıldığı dildir.

Database Alanında Kullanılan Veritabanları

İlişkisel Veri Tabanları (RDBMS):

Verileri tablolar halinde depolar ve tablolar arasındaki ilişkileri kullanarak veri bütünlüğünü sağlar. En yaygın kullanılan veri tabanı türüdür.

Örnekler:

  • MySQL
  • PostgreSQL
  • Oracle
  • Microsoft SQL Server

NoSQL Veri Tabanları:

İlişkisel olmayan veri tabanları, yapılandırılmamış veya yarı yapılandırılmış verileri depolamak için tasarlanmıştır. Büyük veri uygulamaları, gerçek zamanlı veriler ve esnek veri modelleri için idealdir.

Örnekler:

  • MongoDB
  • Cassandra
  • CouchDB

Nesne Yönelimli Veri Tabanları (OODBMS):

Verileri nesneler ve sınıflar şeklinde depolar. Nesne yönelimli programlama dilleriyle entegre çalışabilir.

Örnekler:

  • ObjectStore
  • Versant

Grafik Veri Tabanları:

Verileri düğümler (node) ve ilişkiler (relationship) şeklinde depolar. Sosyal ağlar, önerme sistemleri ve dolandırıcılık tespiti gibi uygulamalar için uygundur.

Örnekler:

  • Neo4j
  • ArangoDB

Cloud Veritabanı :

Bulut bilişim platformları üzerinde çalışan veri tabanlarıdır. Ölçeklenebilirlik, esneklik ve yönetim kolaylığı sağlar.

Örnekler:

  • Amazon RDS
  • Google Cloud SQL
  • Azure SQL
